 This is a submodule of the reacher-\/mp project.\hypertarget{md__r_e_a_d_m_e_autotoc_md18}{}\doxysubsection{\texorpdfstring{What is this repository for?}{What is this repository for?}}\label{md__r_e_a_d_m_e_autotoc_md18}

\begin{DoxyItemize}
\item This is the C language implementation of the Reach protocol stack using Protobufs. It is designed to be included as a submodule in an application.
\item In addition to the core reach functionality, it includes the services such as logging and printing JSON that are used.
\item The silabs directory includes files appropriate for a Silicon Labs EFR32/\+BGM solution.
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md19}{}\doxysubsection{\texorpdfstring{How do I get set up?}{How do I get set up?}}\label{md__r_e_a_d_m_e_autotoc_md19}

\begin{DoxyItemize}
\item Clone or fork the "{}reacher-\/mp"{} super project.
\item Follow the instructions there.
\item You should not need to modify this project for your Reach enabled device. You should be able to make all necessary modifications at the top leve.
